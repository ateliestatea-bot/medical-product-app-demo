# Medical Product App Prototype / Demo

This repository contains a prototype/demo web application for client review.
It is not a production release.

## Overview

- Touch-panel oriented medical product proposal app
- Built with Next.js
- Product, category, and diagnosis content is managed through JSON files
- No admin screen, inquiry feature, or offline support

## Data Files

- `data/products.json`
- `data/categories.json`
- `data/diagnosis.json`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
